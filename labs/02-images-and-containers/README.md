# Docker Images and Containers

## Objectives

- Understand Docker Images
- Understand Docker Containers
- Learn image lifecycle
- Learn container lifecycle
- Pull images from Docker Hub
- Create and manage containers
- Inspect images and containers
- Execute commands inside containers

## Labs

### Lab 1 - Pull an Image

Pull nginx image and verify.

### Lab 2 - Run a Container

Run nginx container.

### Lab 3 - Container Lifecycle

Start, Stop, Restart, Pause and Unpause containers.

### Lab 4 - Inspect Resources

Inspect image metadata and container metadata.

### Lab 5 - Execute Commands

Use docker exec to access running containers.

### Lab 6 - View Logs

Use docker logs to troubleshoot containers.
