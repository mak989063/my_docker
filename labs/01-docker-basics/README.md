# Docker Basics

## Objective

Learn the fundamental Docker concepts and commands required to work with containers.

## Topics Covered

* Docker Architecture
* Images vs Containers
* Container Lifecycle
* Running Containers
* Inspecting Containers
* Container Logs
* Executing Commands Inside Containers

## Hands-on Activities

* Ran Nginx container
* Viewed running containers
* Inspected container metadata
* Accessed container shell using docker exec
* Viewed logs using docker logs

## Key Commands

docker run

docker ps

docker exec

docker logs

docker inspect

docker rm
